# MovieTracker

A full-stack application that allows the user to search up a movie and add it to a list ( watched, watching, watch list). User can also give a movie a rating out of 10 and move the movie in-between lists.

## Features:
* **User Auth**: Secure sign up/login.
* **CURD Ops**: Full Create, Read, Update, and Delete functionality.

## Tech Stack
* **Frontend**: React.js
* **Styling**:
* **Backend**: Node.js/Express
* **Database**: MongoDB

### Prerequisites
* Node.js
* Mongodb (local or Atlas)
